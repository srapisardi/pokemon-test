# Pokémon Test

A small game project inspired by the original Pokémon games, focused on recreating Pallet Town in a simple 2D environment using Pygame.

## Project Overview
This project aims to bring the atmosphere of Pallet Town to life with:
- a top-down town layout
- simple character movement
- building and environment sprites
- a retro-style game feel inspired by the original Pokémon Red and Blue

## Tech Stack
- Python
- Pygame
- Optional: Pygame GUI helpers or sprite-based asset management

## Goals
- recreate the look and feel of Pallet Town
- build a playable scene with movement and interaction
- learn game development fundamentals with Pygame
- experiment with tile-based or sprite-based world design

## Planned Features
- player movement through the town
- houses, paths, and scenery inspired by Pallet Town
- collision handling for buildings and boundaries
- simple NPCs and dialogue
- background music and sound effects

## Getting Started
1. Install Python 3.9+.
2. Install Pygame:
   ```bash
   pip install pygame
   ```
3. Run the game:
   ```bash
   python main.py
   ```

## Project Structure
- `main.py` - main game loop and initialization
- `assets/` - images, sprites, and audio files
- `maps/` - map or level data
- `src/` - game modules and classes

## Notes
This is a prototype-style project and is meant for learning and experimentation rather than a full commercial game.
